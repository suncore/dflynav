Welcome to Dragonfly Navigator

Please read this short documentation, it will make your use of Dragonfly Navigator a lot more efficient.

Opening files or directories

Double click left mouse button or single click right mouse button on a file or directory to open it. If ctrl is pressed when opening a directory, the directory is shown in the opposite pane.

Working with files and directories

Select files or directories in one pane and press a button in the middle. The opposite pane is the “destination”, e.g. for “copy”.

Selection

Press the left mouse + shift to select a range of files.

Press the left mouse + ctrl to select multiple individual files.

When a selection has been started, the pane will pause updating to not disturb the selection process. Pressing refresh will resume updating.

Quick preview

Move the mouse over a file/directory and press the Alt button at the same time. A preview will be shown in the opposite pane.

List of jobs

File operations happen in the background and are called “jobs”. The list at the bottom shows running and completed jobs. Click on a job to see more details. You can expand the job list by dragging the top of the list upwards.

Button mapping summary

LMB = Left Mouse Button

RMB = Right Mouse Button

RMB: Open file or folder. This is not a context-menu!

Doubleclick LMB: Open file or folder.

LMB: Select (with Shift or Ctrl to select range or multiple).

Ctrl + RMB or Ctrl + doubleclick LMB: Open file or folder in opposite pane.

Alt + mouse hover (over file or folder): Preview in opposite pane.
