Opening files or directories

Double click left mouse button or single click right mouse button on a file or directory to open it. If ctrl is pressed when opening a directory, the directory is shown in the opposite pane.

Working with files and directories

Select files or directories in one pane and press a button in the middle. The opposite pane is the “destination”, e.g. for “copy”.

Creating a directory

Type the name of the directory in the directory entry (at the top, above the list of files) and press return.

List of jobs

File operations happen in the background and are called “jobs”. The list at the bottom shows running and completed jobs. Click on a job to see more details. You can drag the top of the list upwards to expand it.

Quick preview

Move the mouse over a file/directory and press the Alt button. A preview will be shown in the opposite pane.

Selection

Press the left mouse + shift to select a range of files.

Press the left mouse + ctrl to select multiple individual files. When a selection has been started, the pane will pause updating to not disturb the selection process. Pressing refresh will resume updating.
