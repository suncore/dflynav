Welcome to Dragonfly Navigator!

Dragonfly Navigator uses a two-pane approach to working with folders, files and other objects. This gives a natural way of specifying source and destination for many operations, e.g. copying or moving files. Selected files/folders are the source and the opposite pane is the destination.

The following is a short overview of the most important things to know about Dragonfly Navigator.

All objects in a single tree

For simplicity, Dragonfly Navigator collects all the information it knows about in a single folder tree. For example, Windows drives are collected under the /Drives path and the user Home folder is in /Home.

Viewing folders

Press the right mouse button on a folder to open it. (Double-click left mouse button also works.) If the Ctrl button is pressed when opening a folder, the folder is shown in the opposite window instead.

Opening files

Press the right mouse button on a file to open it. (Double-click left mouse button also works.)

Working with files and folders

Select files or folders on one side and press one of the buttons in the middle column. If you for example press the copy button, the files will be copied to the opposite folder.

Some operations, like delete, only affect the selected files, not the opposite folder.

The list of available middle buttons change depending on the situation. Sometimes it is not possible to copy objects to the destination. Then the copy button is hidden,

Creating a folder

Type the name of the folder in the folder entry (at the top, above the list of objects). If the folder you typed does not exist, you will be asked if you want to create it.

List of jobs

If you have run actions on files (using the middle column action buttons), the list at the bottom shows a history of operations. Click on one to see more details. You can drag the top of the list upwards to expand it.

Quick preview

It is possible to preview folders and some files (like images). Move the mouse cursor over a file/folder and press the Alt button. A preview of the file/folder will be shown on the opposite side.

Selection tips

Press the Shift button when selecting to select a range of objects.

Press the Ctrl button when selecting to select multiple individual objects.
